## 🇺🇸 English (Description)

**Napa Hardware Advisor** is a deep hardware diagnostic tool designed for high precision. It integrates with the **Global Intelligence Database (Kaggle)** to provide more technical insights than standard system information tools.

### 🌟 Key Features
* **Global Database:** Provides deep insights into Microarchitecture (Lithography nm), Launch Dates, and Integrated Graphics models.
* **Universal Support:** Fully compatible with Desktop PCs, Laptops, MacBooks (Darwin), and Server/Cloud Environments.
* **Detailed Report:** Detailed analysis of RAM slots (Speed/Capacity), Storage health/interface (NVMe/SATA), and Physical Network adapters.
* **Portable & Efficient:** Lightweight executable with no background services.
